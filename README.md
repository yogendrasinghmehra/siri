# siri
siri is voice recognition desktop application built in c#.
siri allows user to perform basic computer operation by verbally saying the command on any supportable computer mike.
siri has built in set of commands, user have to choose from those commands in order to perform operations.
Few example of commands are :

"Hi siri"-- to initialize the application.
"what is today"-- will tell you today date and time.
"good bye siri"-- to quit the application.
